Luminate Online Data Viewer

A desktop application for Windows and OS X providing realtime reporting for Luminate Online.

View on GitHub Download .zip Download .tar.gz

About

The Luminate Online Data Viewer allows organizations to gain insight into the performance of their online marketing campaigns with up-to-the-minute reporting using Luminate Online Web Services.

Download

The Data Viewer is available as an application for 32- and 64-bit Windows and OS X operating systems.
